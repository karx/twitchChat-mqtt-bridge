## Twitch Chat to MQTT bridge
This is a bridge created to publish messaged received in Twitch chat to MQTT 

## Setup
Make sure to rename `sample-config.json` to `config.json` in the root directory with the following:
```
BOT_USERNAME
OAUTH_TOKEN
CHANNEL_NAME
MQTT_URL
```

## Build/Run

```
npm install
node index.js
```
